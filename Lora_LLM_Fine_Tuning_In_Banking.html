Low-rank adaptation based on intrinsic dimensionality

🏗️ LoRA Architecture

🔒 Freeze W₀

Keep pre-trained weights unchanged

🎲 Initialize

A ~ N(0,σ²), B = 0

📈 Scale

ΔWx by α/r

🚀 Deploy

W = W₀ + BA

During Training

  • W₀ remains frozen
  • Only A and B are updated
  • Gradients computed for A, B only
  • Memory efficient

During Inference

  • Merge: W = W₀ + BA
  • No additional latency
  • Same computational cost
  • Easy task switching

💻 LoRA Implementation

class LoRALayer(nn.Module):
    def __init__(self, in_features, out_features, r=4, alpha=1):
        self.r = r
        self.alpha = alpha
        self.lora_A = nn.Parameter(torch.randn(r, in_features))
        self.lora_B = nn.Parameter(torch.zeros(out_features, r))
        self.scaling = alpha / r

    def forward(self, x):
        # ΔW = BA ; scale by α/r
        return (x @ self.lora_A.T @ self.lora_B.T) * self.scaling

🔑 Key Components

  • Matrix A: Random initialization
  • Matrix B: Zero initialization
  • Scaling factor: α/r
  • Frozen base weights
